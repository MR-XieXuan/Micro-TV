# ⭐Micro-TV
Micro-TV 桌面小电视

# ❗️注意事项
* 外壳久置会被氧化发黄，可以喷漆保护，按钮部分可以涂上丙烯颜料更加美观。
* 尽量使用光固化打印！
* 板子打印厚度尽量选择 1.2 mm
